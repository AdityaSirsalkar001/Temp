# Productivity App

A local-first productivity web app with:
- Focus timer (Pomodoro-like)
- Tasks (to-do list)
- Notes
- Day planner

Light/Dark theme toggle is built-in. Optional Supabase sync activates if VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY are provided.

Scripts:
- npm run dev
- npm run build
- npm run preview
